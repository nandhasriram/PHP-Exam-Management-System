# PHP University ERP System

A comprehensive, role-based Enterprise Resource Planning (ERP) system built with Core PHP and MySQL. This system streamlines university operations by integrating multiple administrative and student-facing departments into one centralized platform.

## 🚀 Integrated Modules
* **Examination Management:** End-to-end exam scheduling and result processing.
* **Fee Management:** Financial tracking and payment records.
* **ID Management:** Automated student and staff ID generation.
* **Leave Management:** Application and approval workflow for absences.
* **Scholarship Management:** Application tracking and disbursement records.
* **Student Grievance System:** Ticketing system for student complaints and resolutions.
* **Student Info Management:** Centralized database for core student records.

## 🛠️ Tech Stack
* **Backend:** Core PHP
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
